# 💖 Our Magical Love (v3.1.0)
**A personalized digital sanctuary for Anya.**

This project is a multi-functional personal website designed to celebrate love, manage productivity, and provide a space for relaxation. It combines astronomical data, interactive gaming, and utility tools into a cohesive, "glass-morphism" inspired interface.

## 🌌 New in v3.1.0: The Celestial Update
We've added a brand new **Stargazer** module. You can now explore a procedurally generated night sky, name stars after special moments, and toggle constellations.

## ✨ Features
- **Live Counters:** Real-time tracking of time spent together.
- **World Clocks:** Syncing EST and IST time zones for Micheal & Anya.
- **StudyBuddy:** A Pomodoro-style productivity suite with ambient sounds and session logging.
- **Dreamland:** A relaxation hub with a sleep timer, alarm (with snooze), and lo-fi integration.
- **Stargazer:** Interactive star map with pan/drag and naming capabilities.
- **Minigame:** A custom-themed Pong game with match history and "MBM" (Most Beautiful Memory) logging.
- **Dynamic Sky:** Backgrounds, sun, and moon positions update based on real-world astronomical data (SunCalc).

## 🛠 Tech Stack
- **Frontend:** Vanilla HTML5, CSS3 (Custom Variables/Animations), JavaScript (ES6).
- **APIs:** YouTube IFrame API, SunCalc.js.
- **Analytics:** Custom Sentinel Uplink (Node.js/MongoDB).
- **Hosting:** GitHub Pages.

## 🔒 Privacy
This is a private, personal project. Data logged via analytics is used solely for performance monitoring and safety.

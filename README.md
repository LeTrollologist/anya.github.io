# ✦ Anya's Space (v3.3.0)
**A personal utility hub built for Anya.**

A multi-functional single-page web app combining productivity tools, relaxation, astronomy, and gaming — all in one clean, glass-morphism interface.

## ✨ Features

### 📚 StudyBuddy
Pomodoro-style focus timer with preset sessions (25 / 5 / 15 min) and a **custom timer** — type any duration, pick Focus or Break, and go. Includes ambient sound toggles, motivational messages, and session logging.

### 🌙 Dreamland
Relaxation hub with lo-fi YouTube integration, sleep timer with fade-out, alarm with auto-snooze, and volume control.

### 🌌 Stargazer
Interactive star map built on real astronomical data — 750 stars from the Yale Bright Star Catalogue (V mag ≤ 5.0), IAU constellation stick figures, gnomonic projection, and B-V colour-accurate star rendering. Pan by dragging, click a star to reveal its name.

### 🏓 Pong (Minigame)
Custom-themed Pong with match history, particle effects, and **four difficulty levels:**
- **Easy** — slow AI, wide error margin
- **Medium** — balanced
- **Anya** — named, no hints
- **Hard** — sharp AI, tight margins, maximum speed

### 🌍 World Clocks
Live Dubai 🇦🇪 and India 🇮🇳 clocks, updating every second.

### ⚙️ System Status
Real-time diagnostics for GitHub Pages, Sentinel analytics node, MongoDB, and the check-in tunnel.

## 🛠 Tech Stack
- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES5/6 — no build step)
- **APIs:** YouTube IFrame API, SunCalc.js
- **Analytics:** Sentinel Uplink (Node.js / MongoDB Atlas)
- **Hosting:** GitHub Pages

## 🔒 Privacy
Personal project. Analytics data is used solely for uptime monitoring.
